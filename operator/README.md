# Операторы

    $name = 'Ivan';

    $result = $name == 'Ivan';
    var_dump($result); // bool(true)

    $result = $name == 'Ivan' && $name == 'Petr';
    var_dump($result); //  bool(false)

## Операторы сравнения
- `==`
- `===` - строгое равенство
- `!=`  - не равно
- `!==`
- `>=`
- `<=`
- `>`
- `<`

## Операторы логические
- && - и
- || - или
- !  - не
